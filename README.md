# NumPy Practice 

This repository contains my **NumPy practice notebooks**, created while learning the fundamentals of numerical computing and data manipulation in Python.

All notebooks are practiced and executed on **Google Colab**.

- Creating NumPy Arrays
- Array Attributes (shape, dtype, ndim)
- Indexing & Slicing
- Reshaping Arrays
- Mathematical Operations
- Broadcasting
- Aggregation Functions (sum, mean, min, max)
- Working with CSV / Text Data (`genfromtxt`)
- Real-world numerical examples
